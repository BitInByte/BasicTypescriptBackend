# BasicTypescriptBackend

## A Express Backend Approach

This express backend approach is based on OOP concept. In my opinion, this makes all of the backend code more readable and organised.

## Steps to run it

**Install Modules**

```
    npm install
```

**Watch Ts**

```
    npm run watch:ts
```

**Start server**

```
    npm start
```

## Built With

- NodeJS
- Express
- Typescript
